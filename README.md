# Opis aplikacije

Ideja za aplikacijo je zelo preprosta spletna trgovina, kjer lahko uporabniki nakupujejo izdelke admini pa imajo dostop do razne statistike o prodaji in uporabnikih. Aplikacija je sestavljena tako da se Äim bolj testira znanje iz SQLa.
